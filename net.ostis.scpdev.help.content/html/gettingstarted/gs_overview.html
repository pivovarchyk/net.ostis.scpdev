Общее представление

SCP – это графовый процедурный язык программирования, предназначенный для эффективной обработки однородных семантических сетей. Для кодирования однородных семантических сетей используется sc-код. SCP является sc-языком, следовательно, множество текстов языка SCP является подмножеством всех возможных текстов sc-кода.

Важнейшими особенностями SCP являются:

  • Ориентация на переработку графодинамической ассоциативной памяти. SCP ориентирован на переработку непосредственно семантических сетей, а не структур данных (например, списковых), с помощью которых они кодируются. SCP поддерживает достаточно высокого уровень абстракции для обработки семантических сетей.
  • Представление текстов языка SCP в виде семантических сетей. Не только данные, но и программы представляются в виде семантической сети. Это позволяет использовать тексты языка SCP в качестве данных, а следовательно, перестраиваться программам в ходе выполнения.
  • Возможность интеграции текстов языка SCP с программами на других языках программирования посредствам содержимого sc-узлов.
  • Высокий потенциал распараллеливания процессов переработки информации в графодинамической ассоциативной памяти.

SCP разрабатывается и реализуется в нескольких модификациях, имеющих одинаковый набор выполняемых операций, но различный синтаксис. Система операций SCP и синтаксис представлены в [1, 2, 3].
