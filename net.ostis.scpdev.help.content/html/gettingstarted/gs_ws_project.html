Создание проекта

Выбираем пункт меню File -> New -> Project и затем выбираем тип проекта SC Repository в папке SCP Development (см. рисунок 3.3). Нажимаем на Next. На этой странице мастера по созданию проекта (см. рисунок 3.4) вводим имя проекта и путь к папке с исходными текстами базы знаний "путь"\fs_repo_src (вспомните, что именно сюда мы копировали наш пример).Нажимаем кнопку Finish. Проект создан и должен был быть автоматически собран (см. рисунок 3.5). Обратите внимание, что на вкладке Problems таблица пустая. Т.е. сборка произошла без ошибок. Еще необходимо обратить внимание на две папки в проекте wave_find_path. Они всегда присутствуют в проекте типа SC Repository. В папке fs_repo_src хранятся исходные тексты базы знаний (в том числе и программы), а папку fs_repo является скомпилированной версией fs_repo_src. Содержимое этих двух папок мы рассмотрим в следующем разделе, а сейчас запустим программу-пример.
