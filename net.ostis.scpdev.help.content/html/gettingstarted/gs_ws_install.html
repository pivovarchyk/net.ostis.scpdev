Установка компонентов

Необходимо установить программный модуль sc-core, который представляет собой ядро для обработки sc-текстов. Последнюю версию sc-core можно загрузить "вставить ссылку". Устанавливать sc-core желательно так, чтобы путь не содержал пробелов в именах директорий.При установке необходимо выбрать опцию, которая добавит директорию исполняемых файлов sc-core в переменную среды окружения PATH (рисунок 1). "Вставить рисунок" Инструментальное средство разработки программ на языке SCP представляет собой плагин к платформе Eclipse. Для работы необходимо: 1. установкить Java Development Kit "отсюда" 2. установить eclipse 3. подключить plugins
